# 移动端练手项目
Fork 本项目。完成下面的任务。

## 运行
1. 安装依赖 `yarn install`
1. 运行 `npm run dev`

  * [ ] 列表
    * [ ] 列表下拉无限加载
    * [ ] 下拉刷新列表内容

    * [ ] 图片懒加载
    * [ ] 图片预览(点击图片，显示图片的大图)
  * [ ] 轮播
  * [ ] 加载中
  * [ ] 信息提示(Toast)
  * [ ] 弹出框

    * [ ] 文本框(Filed)，文本域，密码框
    * [ ] 步进器(Stepper)。输入离散数字框
    * [ ] 数字键盘
    * [ ] 开关
